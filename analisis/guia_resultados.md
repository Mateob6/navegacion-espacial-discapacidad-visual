## 3. ANOVA Factorial Completo: 2(CE) × 2(OF) × 2(OT) × 4(MS) sobre E.A

### 3.1 Supuestos

- **Levene (factor entre-sujetos):** F = 5.223, p = .031 → heterogeneidad de varianzas. Se corrobora con análisis robustos (ver §7).
- **Mauchly (esfericidad, factor Modalidad):** W = 0.877, χ² = 3.381, df = 5, p = .642 → esfericidad asumida.
- **Normalidad (Shapiro-Wilk):** Todas las 8 celdas (2 grupos × 4 modalidades) con p > .05 → normalidad asumida en los datos colapsados.
- **Normalidad de residuos del modelo:** W = 0.957, p = .001 → desviación significativa; justifica análisis robustos complementarios.

### 3.2 Modelo global (factor within combinado: 16 niveles)

### 6.2 Error absoluto vs. error con signo

**Hallazgo clave:** Con error absoluto, la interacción CE × MS **desaparece** (p = .322). Esto revela que:
- El efecto principal de grupo es mucho más fuerte con error absoluto (η²p = .477 vs .153).
- La interacción observada con error con signo se debe al **sesgo direccional** (sobreestimación sistemática del NDV), no a diferencias en precisión general.
- Cuando se elimina la información de dirección, las modalidades no modulan diferencialmente los grupos.

#### BF₁₀ para comparaciones de modalidad dentro de cada grupo

**Grupo DV:** Todas las comparaciones BF₁₀ < 0.35 → evidencia moderada-fuerte para H₀ (sin diferencias entre modalidades).

**Grupo NDV:**

| Contraste | BF₁₀ | Interpretación |
|-----------|-------|----------------|
| CIN vs CTR | **76.98** | Evidencia muy fuerte para H₁ |
| ACC vs CTR | **12.69** | Evidencia fuerte para H₁ |
| CIN vs TAC | 1.08 | Evidencia anecdótica |
| TAC vs CTR | 1.45 | Evidencia anecdótica |
| CIN vs ACC | 0.37 | Evidencia moderada para H₀ |
| TAC vs ACC | 0.32 | Evidencia moderada para H₀ |

#### ANOVA bayesiano para la interacción CE × MS

- BIC modelo completo (con interacción) = 1305.7
- BIC modelo reducido (sin interacción) = 1297.0
- **BF₁₀ (interacción) ≈ 0.013** → Evidencia fuerte a favor de H₀

**Nota interpretativa:** El análisis bayesiano sugiere que la interacción CE × MS no es robusta. Esto es consistente con el hallazgo de que la interacción desaparece con error absoluto (§6.2). Sin embargo, los BF₁₀ para comparaciones específicas dentro del grupo NDV (CIN vs CTR = 76.98, ACC vs CTR = 12.69) sí apoyan diferencias genuinas entre modalidades para ese grupo. La discrepancia se explica porque la interacción global incluye las comparaciones nulas del grupo DV, que "diluyen" el efecto.

## 9. Síntesis Narrativa para Guiar la Escritura

### Hilo argumental sugerido para Resultados:

1. **Equivalencia de grupos** (§1): Los grupos no difieren en variables demográficas.

2. **ANOVA completo** (§3): Efecto principal de CE significativo. Interacciones significativas CE × Forma, CE × Tamaño, CE × Modalidad, y de orden superior. Justificar el colapso por forma y tamaño dado que el interés teórico es CE × MS.

3. **ANOVA colapsado CE × MS** (§4): Interacción significativa. Descomponer en efectos simples:
   - DV: sin efecto de modalidad (techo de desempeño).
   - NDV: efecto significativo; Control y Tacto reducen el error vs Cinestésica.
   - En Cinestésica, los grupos difieren (d = −1.19); en Control, son iguales (d = 0.03).

4. **Error angular** (§5): No hay interacción → la modalidad sensorial afecta la estimación de distancia, no de dirección.

5. **Nuevos hallazgos** (§6): Presentar como análisis complementarios:
   - La interacción desaparece con error absoluto → el efecto es sobre el sesgo, no sobre la precisión total.
   - NDV sobreestima sistemáticamente en Cinestésica.
   - NDV es más variable (excepto en Acción).
   - El análisis bayesiano apoya comparaciones específicas dentro del NDV pero no la interacción global.
   - Interacción en tiempo sin correlación velocidad-precisión.

6. **Robustez** (§7): ART ANOVA confirma todos los efectos. Mann-Whitney converge con paramétricos. Potencia moderada.

### Hallazgos que requieren matiz interpretativo:

- **El análisis bayesiano contradice el ANOVA** para la interacción global (BF₁₀ = 0.013), pero apoya comparaciones específicas (BF₁₀ CIN vs CTR en NDV = 76.98). Esto no es contradictorio: el ANOVA bayesiano penaliza modelos complejos, mientras que las comparaciones post-hoc son más focalizadas.
- **El índice de compensación es negativo**, lo que parece contradecir la mejora observada con error con signo. La explicación es que las modalidades externas reducen el sesgo (error con signo → 0) pero no necesariamente el error absoluto total.
- **La condición Control** es la que más reduce el error en NDV, lo cual es contraintuitivo porque no involucra objetos. Discutir en términos de la acción como "andamiaje cognitivo" que estructura la representación espacial.
